# CodeMagic
CodeMagic 是使用C#(WinFORM)编写的代码生成器，特性如下：

- 支持 SqlServer 数据库；
- Model类代码生成
- DAL 类代码生成
- BLL 类代码生成
- MVC Controller CRUD 类代码生成
- AdminLTE UI 代码生成
- 支持批量导出代码到项目文件夹
- 支持 ASP.NET Core 3.1 & MySQL 数据库代码生成 CodeMagic.MySQL

## 开发计划

- 查询分析器生成符合查询条件的Model代码
- 生成MVC Controller 代码
- 一键生成整站Web CRUD代码（UI基于AdminLTE）
- 自定义代码段生成
- 支持流程业务代码生成
